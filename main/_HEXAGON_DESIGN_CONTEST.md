# 🎨 {uteals} Package Hexagon Design Contest

We’re looking for the perfect hexagon logo for the **uteals** package!
Help us create a visual identity that represents our utilities and
extensions for the teal framework.

## About uteals

The `uteals` package provides decorators, transformators, and utility
functions to enhance the [teal
package](https://github.com/insightsengineering/teal) ecosystem. The
package includes:

- **Decorators**:
  [`ggplot_decorator()`](https://phuse-org.github.io/uteals/reference/ggplot_decorator.md),
  [`patchwork_plot_decorator()`](https://phuse-org.github.io/uteals/reference/patchwork_plot_decorator.md),
  …
- **Transformators**:
  [`merge_levels_transformator()`](https://phuse-org.github.io/uteals/reference/merge_levels_transformator.md),
  [`or_filtering_transformator()`](https://phuse-org.github.io/uteals/reference/or_filtering_transformator.md),
  …
- **Module Access Management** wrappers:
  [`extract_modules_to_yaml()`](https://phuse-org.github.io/uteals/reference/extract_modules_to_yaml.md),
  [`keep_by_label()`](https://phuse-org.github.io/uteals/reference/keep_by_label.md),
  [`remove_by_label()`](https://phuse-org.github.io/uteals/reference/remove_by_label.md)

## Design Guidelines

### Required Elements

- Package name: **“uteals”**
- Hexagon format (standard R package hex sticker)
- Colors that complement the teal package palette:
  - Light blue: `#E4EFFF`
  - Medium blue: `#80BAFF`
  - Dark blue: `#4A90E2`
  - Teal green: `#008C72`

### Design Inspiration

- **Utilities theme**: Tools, gears, building blocks
- **Teal connection**: Reference to the teal bird or teal package
  aesthetic
- **Professional**: Suitable for pharmaceutical/clinical research
  context

## How to Submit

1.  **Fork** this repository
2.  **Create** your hexagon design as PNG (recommended size: 181x209
    pixels)
3.  **Add** your design to the `hex_designs/` folder with filename:
    `uteals_hex_[yourname].png`
4.  **Include** a brief description in `hex_designs/README.md`
    explaining your design concept
5.  **Submit** a pull request with title: “Hexagon design submission:
    \[Your Design Name\]”

## Submission Requirements

- Original artwork (no copyrighted elements)
- PNG format, transparent background
- High resolution (minimum 181x209 pixels)
- Follows R package hex sticker conventions
